기안 완료!
❓ 사용법 안내 클릭하여 펼치기
📋 전체 흐름
  1. SEON 포털 또는 BizOffice에서 "영수증 OCR" 카드 클릭 → 자동 로그인
  2. 영수증 파일(JPG, PNG, PDF) 업로드 → OCR 분석
  3. 결과 확인 · 수정 → 현장명 입력
  4. 기안하기 클릭 → BizOffice에서 북마클릿 실행 → 자동 입력
📌 북마클릿 최초 설치 (1회만)
  1. Ctrl+Shift+B (Mac: Cmd+Shift+B) 로 북마크 바 표시
  2. 기안하기 단계에서 파란 버튼 "📋 BizOffice 자동입력"을 북마크 바로 드래그
  3. 설치 완료 — 이후엔 클릭만 하면 됩니다
🖊️ 기안하기 단계 순서
  1. 앱에서 [BizOffice 열기 →] 버튼 클릭 → 새 탭으로 BizOffice 이동
  2. BizOffice: 결재 → 기안하기 → 수입지출결의서 선택 → 확인
  3. 양식 팝업이 열리면 북마클릿 클릭
  4. 앱에 표시된 4자리 코드 입력 (30분 유효)
  5. 폼 자동 입력 완료 → 내용 확인 후 BizOffice에서 기안
⚠️ 자주 묻는 질문
Q. 로그인 화면이 나와요
→ 반드시 SEON 포털 또는 BizOffice에서 접속해야 합니다. 주소창에 직접 입력하면 차단됩니다.
Q. OCR 결과가 이상해요
→ 내역 확인 단계에서 직접 수정할 수 있습니다. 금액·거래처·적요 모두 편집 가능합니다.
Q. 코드가 만료됐다고 해요
→ 30분이 지난 경우입니다. 앱으로 돌아와 기안하기를 다시 클릭하면 새 코드가 발급됩니다.
Q. 자동입력이 안 돼요
→ 수입지출결의서 양식이 완전히 열린 뒤 북마클릿을 클릭해주세요.
